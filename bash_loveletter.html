Use command: cd "current directory". This command helps to switch in and out of directories (folders)
cd ~ = takes to user directory
cd - = takes back to most recent directory


Use command: pwd "present working directory". This shows the current directory that your working in the terminal on


To make a file, use command: touch and then the name of the file you wish you create. *hit tab for a file suggestion to finish the file name*

To get a list of contents within the directory, use command ls This will list all the files within the folder.


mkdir= makes a new directory (folder)


mv= used to rename files




cp= command to copy



rm= deletes FILES. note: deleting from terminal gets rid of file forever (theres no going back)



rmdir= deletes directories that are empty

rm -r= removes directories even if they are not empty
